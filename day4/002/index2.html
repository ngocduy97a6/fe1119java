có thể gộp
        padding-top, padding-bottom , padding-left, padding-right
        vào 1 thuộc tính duy nhất là padding
        ví dụ
        padding: 20px;
        quy tắc  giống hệt margin
TH1 : padding: 30px; => padding-top: 30; padding-bottom: 30; padding-left: 30; padding-right: 30;
TH2 : padding: 30px 10px 6px 20px; => padding-top: 30; padding-bottom: 6; padding-left: 20; padding-right: 10;
TH3 : padding: 5px 9px 12px; => padding-top: 5; padding-bottom: 12; padding-left: 9; padding-right: 9;
TH4 : padding: 10px 20px; => padding-top: 10; padding-bottom: 10; padding-left: 20; padding-right: 20;
